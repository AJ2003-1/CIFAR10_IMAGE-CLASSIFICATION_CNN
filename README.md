# CIFAR10_IMAGE-CLASSIFICATION_CNN
This project applies a Convolutional Neural Network (CNN) for image classification on the CIFAR-10 dataset. The model is trained to recognize 10 object categories using deep learning concepts such as convolution, pooling, and dense layers. Evaluation metrics and accuracy results are included.

## Description
This project trains a Convolutional Neural Network (CNN) on the CIFAR-10 dataset for image classification. The process includes data preprocessing, model training, and evaluation.

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Dataset
- Dataset: CIFAR-10
- Classes: 10
- Source: Available via keras.datasets

## Results
- Training Accuracy: 75%
- Test Accuracy: 69%

## How to Run
Install dependencies:
pip install tensorflow numpy matplotlib

Open the notebook and run cells.
